# 🔑 API 키 관리

간단하게 API 키를 등록하고, 작동 여부를 확인하세요!

## ✨ 기능

1. **API 키 등록** - 여러 개의 API 키를 한 곳에서 관리
2. **유효성 검증** - 클릭 한 번으로 키가 실제로 작동하는지 확인
3. **간편 삭제** - 안 쓰는 키는 바로 삭제

## 🚀 사용법

1. **브라우저에서 `api-key-manager.html` 열기**

2. **API 키 등록**
   - 키 이름 입력 (예: "내 Gemini 키")
   - API 키 값 입력
   - API 타입 선택 (Gemini, OpenAI 등)
   - "➕ API 키 추가" 클릭

3. **유효성 검증**
   - 개별 검증: 각 키의 "✓ 유효성 검증" 버튼
   - 일괄 검증: "🔍 모든 키 유효성 검증 실행" 버튼

4. **키 삭제**
   - 안 쓰는 키는 "🗑️ 삭제" 버튼으로 제거

끝!

## 📊 지원 API

- ✅ Google Gemini API
- ✅ OpenAI API
- ✅ Anthropic Claude API
- ⚙️ 기타

## 🛡️ 보안

- API 키는 브라우저의 LocalStorage에 저장됩니다
- 공용 컴퓨터에서는 사용 후 데이터를 삭제하세요
- 프로덕션 환경에서는 환경 변수 사용을 권장합니다
